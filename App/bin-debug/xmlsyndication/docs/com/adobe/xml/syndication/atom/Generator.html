com.adobe.xml.syndication.atom

Class Generator

public class Generator

Class that represents an generator element within an Atom feed. The tag identifies the agent used to generate a specific feed.

See also
http://www.atomenabled.org/developers/syndication/atom-format-spec.php#rfc.section.4.2.4


Property Summary
uri:String
An IRI reference which when dereferenced, the resulting URI (mapped from an IRI, if necessary) SHOULD produce a representation that is relevant to that agent.
value:String
A string that is a human-readable name for the generating agent.
version:String
Identifies the version of the generating agent.
