com.adobe.xml.syndication.generic

Class RSS10Item

All implemented interfaces
IItem

public class RSS10Item

Class that abstracts out the specific characteristics of an RSS 1.0 item into a generic Item. You create an instance using an Item10 object, then you can access it in a generic way.
