length Property

public var length:Number

Indicates an advisory length of the linked content in octets / bytes. This is a hint about the content length of the representation returned when the IRI in the href attribute is mapped to a URI and dereferenced. If the length attribute is not present in the original Atom document, the length will return NaN.

type Property

public var type:String

A String that specifies the link's advisory media type. This is a hint about the type of the representation that is expected to be returned when the value of the href attribute is dereferenced.
