IRSS

public class RSS10
extends NewsParser

Class for parsing RSS 1.0 feeds. Note: the RSS10 class cannot be use to parse RSS versions 0.91, 0.92, or 2.0. Use the RSS20 class for all other versions of RSS other than 1.0.

See also
http://web.resource.org/rss/1.0/
